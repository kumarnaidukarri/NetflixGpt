Netflix Gpt

Setup:-
i) Vite build tool:
    npm create vite@latest NetflixGpt
    cd NetflixGpt
    npm install
    npm run dev
ii) TailwindCss library:
   npm install tailwindcss @tailwindcss/vite    
   configure vite plugin and import 'tailwind' in your css.
iii) React-Router library:
     npm i react-router 
     import { createBrowserRouter } from "react-router";
     import { RouterProvider } from "react-router/dom";   


Design :-
1) Home Page:
    Signin form, Signup form
    Redirect to browse page 
2) Browse Page:
   Header
   Main Movie - trailer in background, title, description
   Movie Suggestions - Movie Lists(horizontal scrolls)
3) NetflixGpt Page:
   Search Bar
